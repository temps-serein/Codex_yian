CORE FEATURES

智能识别系统环境

自动识别 Windows、macOS、Linux、Android 等系统版本，检测 CPU、GPU、磁盘、网络、权限和已有软件环境。

每个 Agent 只负责一个明确任务，例如安装 Steam、配置 PaddleOCR、修复 VC 运行库，降低复杂度和风险。

权限请求与控制
